This project is from a long time ago when I was teaching myself how to code with Arduino. I used a hall-effect sensor on the pedals which activated each time a magnet on the pedals went past. The brake levers had light sensors on which controlled some parameters on an additive synth I wrote in SuperCollider. 

Finally, because I thought it would be funny, I linked the thing up to a sega genesis rom and spent the afternoon irritating my housemates.
